Para la ejecución del código es necesario tener todos los archivos del zip en la misma carpeta.
Se ejecuta el código llamado "principal.m"

1.Elcódigo "principal.m" tiene una variable t0 vector que indica el punto de inicio del 
decenso de gradiente.

2.Se pueden modificar los valores de maxiter,m minibatch,epsilon,lr y maxerror con
el escalar adecuado

3.Es necesario que la variable "method" sea una string válida, se puede escoger el método 
quitando el # a la linea correspondiente.

3.Si el punto theta inicial es de grado 2 se obtienen las gráficas solicitadas en el punto 
  3 y 4 de la tarea, de lo contrario no se muestran.

4.Las gráficas solicitadas en el punto 5 y 7 se muestran para cualquier método seleccionado.
